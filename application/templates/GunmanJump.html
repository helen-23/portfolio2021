    Background

    This video game was inspired from Doodle Jump and Gun Mayhem. I wrote the game in Grade 12 using Actionscript3, and since this is a personal project, I have uploaed all source code on GitHub. Portal to the code is here. Below are two short demos I recorded for the game (single player and multiplayer mode, respectively).

    Game Play

    In GunmanJump, your goal is to guide your character up a never-ending series of platforms without falling, while also avoiding bullets that will nudge the you off the platforms. In single player mode, these bullets come from a series of cannons positioned on the platforms, while in multi-player mode, players have guns to battle against each other.

    Design

    The main page provides two game options: single player mode and multi-player mode.

    In single player mode, there are randomly positioned cannons on parts of the platforms, and they will periodically shoot bullets in a fixed direction, you need to dodge the bullets while moving your way up.

    In multiplayer mode, the game can support up to three players, and thier task is to nudge their enemies off the platform with a gun.

    A fun feature of the game I have decided to implement is the customization of characters. At the start of every game, it prompts the player(s) to enter a name and pick a costume. The interface will also ask the player(s) to specify the number of lives for each player in this round.

    Each time when a player shoots someone off the platforms in multiplayer mode, that player will be awarded with points. When the game ends with just one player left standing, you will see a leaderboard with the history scores of all existing players.
